2.1. The Open Data Protocol (OData)¶

This module defines functions and classes for working with the OData, a data access protocol based on Atom and Atom Pub: http://www.odata.org/

This module deals with version 2 of the protocol.

2.1.1. Reference¶

2.1.1.1. Elements¶

class pyslet.odatav2.Client(serviceRoot=None)¶

Bases: pyslet.rfc5023.Client

An OData client.

Can be constructed with an optional URL specifying the service root of an OData service.

feeds = None¶

a list of feeds associated with this client

feedTitles = None¶

a dictionary of feed titles, mapped to collection URLs

pageSize = None¶

the default number of entries to retrieve with each request

None indicates no restriction, request all entries.

AddService(serviceRoot)¶

Adds the feeds defined by the URL serviceRoot to this client.

RetrieveFeed(feedURL, odataQuery=None)¶

Given a feed URL, returns a pyslet.rfc4287.Feed object representing it.

RetrieveEntries(feedURL, odataQuery=None)¶

Given a feed URL, returns an iterable yielding Entry instances.

This method uses the pageSize attribute to set the paging of the data. (The source may restrict the number of return values too). It hides the details required to iterate through the entire list of entries with the caveat that there is no guarantee that the results will be consistent. If the data source is being updated or reconfigured it is possible that the some entries may be skipped or duplicated as a result of being returned by different HTTP requests.

AddEntry(feedURL, entry)¶

Given a feed URL, adds an Entry to it

Returns the new entry as returned by the OData service. entry must be an orphan element.

RetrieveEntry(entryURL)¶

Given an entryURL URL, returns the Entry instance

__getitem__(key)¶

Enables Entry to be suffixed with, e.g., [‘PropertyName’] to read property values.

Returns the value of the property with key. The type of the value will be the type specified by the type attribute on the property.

__setitem__(key, value)¶

Enables Entry to be suffixed with, e.g., [‘PropertyName’] to set property values.

Sets the property key to value. The type of value should be compatible with the type expected by the Property type (as indicated by the type attribute). When setting new properties value the type is not specified and is treated as text - see Property.SetValue() for more information.

Adds a link with name linkTitle to the entry with linkURI.

2.1.1.2. Base Classes¶

class pyslet.odatav2.Property(parent, name=None)¶

Bases: pyslet.odatav2.ODataElement

Represents each property.

The OData namesapce does not define elements in the dataservices space as the elements take their names from the properties themselves. Therefore, the xmlname of each Property instance is the property name.

GetValue()¶

Gets an appropriately typed value for the property.

Overloads the basic GetValue() implementation to add support for type attribute.

SetValue(value)¶

Sets the value of the property using the conversion indicated by the type attribute, if present

When creating new entries you won’t necessarily know the require type, in which case the value is simply converted to a string using the default string conversion method defined by the python object in question.
