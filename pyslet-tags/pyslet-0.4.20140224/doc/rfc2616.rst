Hypertext Transfer Protocol (RFC2616)
=====================================

.. py:module:: pyslet.rfc2616

This module defines functions and classes for working with HTTP as defined by
RFC2396: http://www.ietf.org/rfc/rfc2616.txt

The purpose of this module is to expose some of the basic constructs (including
the synax of protocol components) to allow them to be used normatively in other
context.  The module does contain a functional HTTP client.
