Duration

Duration instances hold periods measured in either:

  • years, months, days, hours, minutes and seconds; or
  • weeks.

The consituent parts are called components. An instance that uses the first form is referred to as being in calendar mode, in the second form week mode.

A Duration instance in calendar mode may have reduced precision by the omission of a value for the seconds, the seconds and minutes, the seconds, minutes and hours, and so on. An instance in calendar mode always has a value for the number of years, though this may be omitted when the instance is represented as a string if it is zero.

SetZero(self)

Sets the instance to calendar mode with all components set to 0.

SetCalendarDuration(self,years,months,days,hours,minutes,seconds)

Sets the instance to calendar mode with the given component values.

GetCalendarDuration(self)

Returns the tuple of (years,months,days,hours,minutes,seconds) representing the duration when in calendar mode or raises an error.co

SetWeekDuration(self,weeks)

Sets the duration to week mode with the given value.

GetWeekDuration(self)

Returns the number of weeks when in week mode or raises an error.

SetFromString(self,durationStr)

Sets the duration from the string representation in durationStr.

GetString(self,truncateZeros=0,ndp=0)

Returns a string representation of the duration, formatted according to the standard.

Leading components with zero values can be omitted by setting truncateZeros, in the case of truncated zero durations the least significant component is printed to satisfy the requirement that at least one value/designator pair must be given.

The magnitude of ndp determines the number of decimal places printed after the value of the least significant component if that component was originally specified as a decimal. If ndp is positive the prefered decimal point character, the comma, is used. If ndp is negative then the alternative decimal point character, the period, is used instead. Note that fractional components of durations are always rounded towards zero.
